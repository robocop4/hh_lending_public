# About Сompany
VSYS Host is a hosting provider (Virtual Systems LLC) founded in 2009. The company offers VPS, dedicated servers, shared hosting, cloud solutions, and infrastructure for high-traffic projects.

# Bug Bounty Key Points

## Eligible Vulnerabilities
The program mainly focuses on vulnerabilities that:
- lead to personal data disclosure;
- allow unauthorized access;
- expose sensitive system information.

To qualify for a reward, you must provide:
- a working Proof of Concept (POC);
- clear reproduction steps;
- proof of real impact.

Minor or low-impact bugs may receive acknowledgment only, without monetary rewards.

---

# Main Reward Targets ($5000)

You must complete **one** of the following tasks:

## 1. File Vulnerability
Read the `bounty` variable from the file:

```php
bounty-hunter.php
```

## 2. Database Vulnerability
Extract data from the table:

```sql
bounty_hunter
```

Successful exploitation + valid report with POC and reproduction steps =

```text
$5000 reward
```

---

# If the Main Goal Is Not Achieved

Even if the primary objective is not completed, you may still contact them if you:
- discovered a serious vulnerability;
- gained unauthorized access;
- accessed sensitive or secret information.

A custom reward may still be discussed.

---

# Responsible Disclosure Rules

Do NOT:
- abuse vulnerabilities beyond verification;
- disclose or share user data;
- publish findings publicly.

---

# Non-Eligible Issues

No rewards for:
- low-impact vulnerabilities;
- already known issues;
- vulnerabilities without security impact;
- bugs that do not expose data or grant access.

---

# Program Focus

The program rewards:
- real critical vulnerabilities;
- proven access to sensitive data;
- practical exploitation, not theoretical issues.
