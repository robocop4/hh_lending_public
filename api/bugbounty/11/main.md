# CryptoAuthLib

## Product Overview

CryptoAuthLib is an open-source cryptographic library developed by Microchip Technology to provide a consistent API for interacting with Microchip CryptoAuthentication™ devices. It supports secure key storage, cryptographic operations, device authentication, and secure boot use cases across embedded systems, IoT devices, and secure elements. The library is designed to work across multiple platforms and operating systems, enabling developers to integrate hardware-backed security into their applications with minimal effort.

Repository: https://github.com/MicrochipTech/cryptoauthlib

## Bug Bounty & Security Policy Summary

We take the security of CryptoAuthLib very seriously. Security vulnerabilities should be reported directly to the **Microchip Product Security Incident Response Team (PSIRT)**, which is responsible for handling potential security issues in Microchip products, including related hardware, software, firmware, and tools.
